## 実装が難しかった点と対策
### 難しかった点
InputSystemを用いて、キーを基準にしたキーバインドの変更を可能にした点
### 対策
InputSystemが用意しているキーバインドは、各操作を基準にしたキーバインドです。  
よって、内部的にはスクリプタブルオブジェクトなどを用いて操作を基準にバインドを設定しています。
